Get Started in Seconds

01

Clone & Install

Get the code and install dependencies using uv for speed.

git clone https://github.com/mrorigo/repo-roast.git cd repo-roast uv pip install -r requirements.txt
02

Configure API

Set up your LLM credentials in a simple .env file.

echo "OPENAI_API_KEY=sk-..." > .env
03

Run the Roast

Choose your roast flavor and let it cook.

uv run repo_roast.py --type maintainability-refactoring-plan --output report.md
